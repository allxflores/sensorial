By using two complementary colors, they are able to have a strong contrast with each other. When a red overlay is on top of the text or illustration it blends with the red color. The green overlay blends with the green illustration or text and is able to display the reds.

Mediante el uso de dos colores complementarios , que son capaces de tener un fuerte contraste entre sí. Cuando una capa roja es en la parte superior del texto o ilustración que se mezcla con el color rojo . La superposición de verde se mezcla con la ilustración o texto y es capaz de mostrar los rojos.
